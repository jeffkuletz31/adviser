HTML Structure

Page Structure

On the screenshot above you can see the basic page structure. Feel free to edit any of the page by opening its corresponding .html file with a code editor.

Social icons

To add a link to your social icon paste the necessary URL into the <a href=> tag. See the example below:

<a href="#" target="_blank" class="social_icons social_tumblr icons"> <span class="icon-tumblr"> </span> <span class="sc_socials_hover icon-tumblr"> </span> </a>

Typography

Image alignment can be changed by adding one of the values (alignleft/alignright) into the <figure class=> tag:

<figure class="sc_image middle alignright sc_image_shape_square"> <div> <img src="images/Depositphotos_8849452_original.jpg" alt=""> </div> </figure>

Highlight



To highlight an element add one of the values below into the <span class=> tag:

<span class="sc_highlight sc_highlight_style_3">vitae dicta sunt</span>

Skills

You can customize the skills bar by changing width and data-ed values as well as their styles. Check below screenshot.

<div class="sc_skills_item sc_skills_style_1 inited"> <div class="sc_skills_count" style="width: 98%;"> </div> <div class="sc_skills_total" data-start="0" data-stop="98" data-step="1" data-max="100" data-speed="11" data-duration="1078" data-ed="%">98%</div> </div>

Contact form



Below is the file responsible for contact form functionality:

root directory.../include/contact-form.php root directory.../js/custom/_form_contact.js

Your email address should be specified in the contact-form.php file. See the code below.

Google map



Files responsible for Google map functionality:

root directory.../js/custom/core.googlemap.js

root directory.../js/shortcodes.js

root directory.../contact-us.html

Address/coordinates should be specified in the contact-us.html file. It also depends on what Google map you are going to use. Find the id below.

class="sc_googlemap sc_googlemap_style_2"

<div id="sc_googlemap_1" class="sc_googlemap sc_googlemap_style_2" data-address="New-York" data-description="New-York" data-latlng="" data-zoom="16" data-style="greyscale" data-point="images/map-marker1.png"></div>
